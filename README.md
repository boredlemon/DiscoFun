## What's DiscoFun?

<a href="https://github.com/DiscoFun" target="_blank">DiscoFun</a> is a new panel that allows you to message, manage and gain statistics inside of your discord bot.
<br/>

<!-- Features -->

## ‚ú® Features

- üìä **Discord Bot Statistics:** Gain full insights and details of your discord bot with server and bot information.
- üí¨ **Simple Messaging Interface:** DiscoFun allows you to easily message users and chat within servers using your discord bot. The panel also has a light and dark theme to suit your needs!
- üîí **Security:** DiscoFun does not run on any servers and can be self hosted. All bot details are stored locally and is completely secure.
- üëÄ **Opensource:** DiscoFun is open source on GitHub so you can see all the code behind the project.

<br/>

<!-- Getting Started -->

## üõ† Getting Started

Simply open `index.html` to open the web version of DiscoFun or use the control panel to open the desktop application by running `DiscoFun.cmd`.
<br/>

<!-- Installation -->

### **Step 1:** Prerequisites
To use the **desktop application** of DiscoFun, you will need to have the following installed.

- Node.js
- npm

[Instructions](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

> ‚ö†Ô∏è **NOTE**: To use the **web version** of DiscoFun, you will need to set a custom User Agent.

[Setting a custom User Agent](./docs/prerequisites/prerequisites.md)
### **Step 2:** Setup / Installation

1. Clone the repository

```sh
git clone https://github.com/OverjoyedToast/DiscoFun
```

> ‚ö†Ô∏è **NOTE**: The updater only works if you have cloned the repository.

2. Open `login.html` for the web application or open `DiscoFun.cmd` to run the desktop application.

3. Login to your discord bot using your discord bot token.

4. Start using DiscoFun!
